flattop function

Array flattop (int M, { bool sym: true })

Return a flat top window. Parameters

M : int Number of points in the output window. If zero or less, an empty array is returned. sym : bool, optional When true (default), generates a symmetric window, for use in filter design. When false, generates a periodic window, for use in spectral analysis. Returns

w : Array The window, with the maximum value normalized to 1 (though the value 1 does not appear if M is even and sym is True). Notes

Flat top windows are used for taking accurate measurements of signal amplitude in the frequency domain, with minimal scalloping error from the center of a frequency bin to its edges, compared to others. This is a 5th-order cosine window, with the 5 terms optimized to make the main lobe maximally flat. 1_ References
