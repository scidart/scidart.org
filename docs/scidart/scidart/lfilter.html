lfilter function

Array lfilter (Array b, Array a, Array x)

Filter data along one-dimension with an IIR or FIR filter. The filter is a direct form II transposed implementation of the standard difference equation. Parameters

b : Array The numerator coefficient vector sequence. a : Array The denominator coefficient vector sequence. x : Array An input array. Returns

y : Array Filtered x Array.

Implementation

Array lfilter(Array b, Array a, Array x) {
  Array result = Array.fixed(x.length);
  for (int i = 0; i < x.length; ++i) {
    double tmp = 0.0;
    for (int j = 0; j < b.length; ++j) {
      if (i - j < 0) continue;
      tmp += b[j] * x[i - j];
    }
    for (int j = 1; j < a.length; ++j) {
      if (i - j < 0) continue;
      tmp -= a[j] * result[i - j];
    }
    tmp /= a[0];
    result[i] = tmp;
  }
  return result;
}
